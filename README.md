# [miyashita.com](https://miyashita.com)

## Requirement
- Node.js
- Ruby

## Install
```bash
yarn install
bundle install --path vendor/bundler
```

## Usage

### Watch
```bash
npm run watch
```

### Build
```bash
npm run build
```

## Contribute
If you change any code, you must run `npm run build` before commit.
